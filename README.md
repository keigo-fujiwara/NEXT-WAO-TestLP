# NEXT-WAO-LP

`N-E.X.T. ハイスクール構想` の紹介LPです。  
`index.html` 単体で動作し、GitHub Pages 公開を想定しています。

## Files

- `index.html`: 公開用LP本体
- `CURSOR_PROMPT.md`: Claude向け指示書を Cursor 向けに再構成した版
- `.github/workflows/pages.yml`: GitHub Pages 自動公開用ワークフロー

## GitHub Pages

このリポジトリは `GitHub Actions` 方式で Pages 公開する構成です。初回のみ、GitHub のリポジトリ設定で `Settings > Pages > Build and deployment > Source` を `GitHub Actions` に変更してください。
