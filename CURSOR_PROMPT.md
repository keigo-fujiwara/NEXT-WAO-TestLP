## Cursor向け 制作プロンプト

### 役割
あなたは経験豊富なWebデザイナー兼フロントエンドエンジニアです。  
官公庁・教育機関向けのB2G水準に耐える高品質なプロモーションサイトを、`Cursor` 上で実装してください。

### 目的
以下の要件に基づき、**「N-E.X.T. ハイスクール構想」** のコラボレーション紹介サイトを、**単一ファイルの `index.html`** として制作してください。  
最終成果物は **GitHub Pages で公開できる静的LP** とし、プレゼン用途でも使える完成度を目指します。

### 実装方針
- `index.html` 1枚で完結する構成にする
- CSSは `style` タグ内にインライン記述する
- JavaScriptは `script` タグ内にインライン記述する
- 外部ライブラリは必要最小限にし、`Vanilla JS` を採用する
- レスポンシブ対応はモバイルファーストで行う
- GitHub Pages でそのまま公開できる静的構成にする

### 前回からの必須変更
1. 主体は **K&Eホールディングス**
2. **WAOは後方支援パートナー** として明示
3. **文部科学省の公式基金** としての信頼感を強調する
4. **3名の人物紹介** を必ず掲載する
5. 読者ターゲットは **都道府県教育委員会担当者・学校法人理事・校長先生**

### 必須セクション
1. ヒーローセクション
2. N-E.X.T. 構想とは
3. K&Eホールディングスが提供する8つのサポート
4. 1年間の伴走サポートフロー
5. パートナー紹介
6. 人物紹介
7. 実績校
8. お問い合わせ

### デザイン要件
- カラーパレットは指定変数に準拠する
- フォントは `Noto Sans JP`
- 固定ヘッダー、スムーススクロール、アクティブセクション表示を実装する
- カウントアップ、スクロールフェードイン、ホバー演出を入れる
- 画像が読めない場合の代替UIを用意する

### 品質条件
- HTML構文エラーがない
- 全セクションに `id` を付与する
- 全画像に `alt` を付ける
- モバイルメニューが動作する
- カウントアップが動作する
- 問い合わせフォームにバリデーションがある
- 文部科学省・K&E・WAOへの外部リンクを設置する

### 公開条件
- GitHub Pages で公開できるようにする
- プレゼン時にそのまま見せられる見た目に仕上げる
- 必要であれば Pages 用のワークフローも追加する
